Class: Configuration

Configuration

new Configuration()

Defines a set of configuration values used to connect to an Oracle NoSQL Database, this is just the constructor with no parameters, once this object is created feel free to change any parameter.
Properties:
Name Type Description
server string defines the address or host name to be connected,
if multiple hosts are going to be used then use a string array with the host names,
all hosts are in the form host:port
storeName String the store name to be used
helperHosts String | Array
readZones String | Array
securityProperties SecurityProperties used for the connection
socketOpenTimeout Number a number in milliseconds to specify the timeout for the socket to connect
socketReadTimeout Number a number in milliseconds to specify the timeout for the socket to read
defaultConsistency Consistency indicates the consistency used by default
defaultDurability Durability indicates the durability used by default
startLocalProxy bool indicates if the client should start a proxy
the String proxy address to connect to if no proxy is started by the client
