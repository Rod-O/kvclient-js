Class: WriteOptions

WriteOptions

new WriteOptions() → {WriteOptions}

WriteOptions is passed to store operations that can update the store to specify non-default behavior relating to operation durability and timeouts.
Properties:
Name Type Description
durability Durability Defines the durability characteristics associated with a standalone write (put or update) operation
returnChoice ReturnChoice Specifies whether to return the row value, version, both or neither.
timeoutMs Number The upper bound on the time interval for processing the operation.
